# AI-Based Real-Time Surveillance System

Standalone project built on YOLO for intelligent edge surveillance.

## Overview
Developed a real-time computer vision system for object detection, segmentation, fire detection, motion detection, and intruder alerting.

## Key Features & Achievements
- Deployed on NVIDIA Jetson Nano with live CCTV input for low-latency edge processing  
- Utilized YOLO for accurate, real-time inference  
- Integrated alerts for fire/motion/intruder events  

## Technologies
- **Hardware**: Jetson Nano  
- **Computer Vision**: YOLO, OpenCV  
- **Programming**: Python  
- **Tools**: Linux, Docker
